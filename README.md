# 大学の 貸出申請・管理アプリ

Djangoで作成した、学内備品の貸出申請・返却管理を行うための業務アプリケーションです。  
PC、VRゴーグル、周辺機器などの貸出状況をWeb上で管理できます。

## 主な機能

- ログイン / ログアウト
- 備品一覧の表示
- 備品詳細の表示
- 備品の種類・状態による絞り込み
- 貸出申請
- 貸出完了画面の表示
- 貸出中一覧の確認
- 返却処理
- 貸出履歴の確認
- 返却期限超過の表示
- 管理者による備品の追加・編集・削除
- 管理者によるユーザー管理
- ユーザーごとの学籍番号管理
- 学年ごとのグループ管理


## 使用技術

- Python
- Django
- SQLite、ローカル開発環境
- PostgreSQL、Render本番環境ではNeonを使用
- Gunicorn
- WhiteNoise
- Render
- Neon

## ユーザー種別

### 一般ユーザー

一般ユーザーは、主に学生を想定しています。

できること:

- 備品一覧を見る
- 備品詳細を見る
- 貸出申請を行う

できないこと:

- 管理画面に入る
- 備品を追加・編集・削除する
- 他ユーザーの個人情報を見る
- 貸出履歴を管理する

### 管理者ユーザー

管理者ユーザーは、`is_staff` がオンのユーザーです。

できること:

- 管理画面に入る
- 備品を追加・編集・削除する
- 貸出中一覧を見る
- 返却処理を行う
- 貸出履歴を確認する
- ユーザー情報を管理する
- 学籍番号を登録・変更する
- グループを作成・編集する

### スーパーユーザー

スーパーユーザーは、`is_superuser` がオンのユーザーです。  
すべての権限を持つ最上位の管理者です。
